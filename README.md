## Requirements

iOS 13
Swift-5

## Features Included
- Animations
- Network services
- Cache for offline usage
- Light and Darkmode
- Loading without locking the screen

## Third-party libraries
- Nuke: used to download images faster and smothly
- Realm: used to make data cache for offline usage
